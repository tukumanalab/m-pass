最終更新日: 2025年11月7日

1. サービス概要

[アプリ名]は、Tukumana Labが運営する会員管理システムです。 本サービスは、会員の皆様に安全で便利なチェックイン機能とメンバー管理機能を提供します。

2. 収集する情報

基本情報

  • 名前、メールアドレス
  • パスワード（暗号化して保存）
  • 会員ID、登録日時
  • チェックイン/チェックアウト記録

技術情報

  • IPアドレス、ブラウザ情報
  • アクセス日時、利用状況

3. 情報の使用目的

  • 会員認証とアカウント管理
  • チェックイン/チェックアウト機能の提供
  • 確認メールやパスワードリセットメールの送信
  • システムの安全性確保と改善

4. Google サービスとの連携

Gmail API の使用

本サービスは、Gmail APIを使用してメール送信機能を提供します：

  • 目的: 会員登録確認メール、パスワードリセットメールの送信
  • アクセス範囲: 当団体が作成するメール内容の送信のみ
  • 重要: ユーザーのGmailアカウントやメール内容にはアクセスしません
  • 認証: OAuth 2.0による安全な認証
  • 権限: 最小限の権限（メール送信のみ）

データの取り扱い: Gmail API経由で送信されるメールは、送信後に当団体のシステムには保存されません。 Googleのプライバシーポリシーにも準拠しています。

5. 情報の共有

以下の場合を除き、個人情報を第三者に提供することはありません：

  • ユーザーの同意がある場合
  • 法令に基づく要請がある場合
  • 生命、身体または財産の保護のために必要な場合

サービスプロバイダー

  • Google LLC: メール送信サービス（Gmail API）
  • クラウドホスティングプロバイダー: サーバー運用のため

6. データの保護

  • 暗号化: パスワードとデータ通信の暗号化
  • アクセス制御: 適切な権限管理
  • HTTPS通信: すべての通信の暗号化
  • 国内サーバー: 日本国内での安全な保管

7. ユーザーの権利

  • アクセス権: 個人情報の確認
  • 修正権: 登録情報の変更
  • 削除権: アカウントの削除要求
  • データポータビリティ: データのエクスポート

8. データの保持期間

  • アクティブアカウント: サービス利用期間中
  • 非アクティブアカウント: 最終ログインから2年間
  • ログデータ: 6ヶ月間

本プライバシーポリシーは、日本国の個人情報保護法に準拠しています。

Google OAuth 認証について

本サービスがGoogle OAuth認証をリクエストする理由は、 Gmail API を使用したメール送信機能のためです。 これにより、会員登録の確認メールやパスワードリセット機能を安全に提供できます。

ユーザーの個人的なGoogleアカウント情報にはアクセスせず、 メール送信に必要な最小限の権限のみを使用します。
